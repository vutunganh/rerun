Module item_heading_with_breadcrumbs

Expand description

The heading of each item in the selection panel.

It consists of a blue background (selected background color), and within it there are “bread-crumbs” that show the hierarchy of the item.

A > B > C > D > item

Each bread-crumb is just an icon or a letter. The item is an icon and a name. Each bread-crumb is clickable, as is the last item.

The bread crumbs hierarchy should be identical to the hierarchy in the either the blueprint tree panel, or the streams/time panel.
