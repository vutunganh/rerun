Module wgpu_resources

Source
Expand description

Wgpu resource pools are concerned with handling low level gpu resources efficiently.

They facilitate easy creation and avoidance of unnecessary gpu allocations.

This is in contrast to the crate::resource_managers which are concerned with higher level resources that arise from processing user provided data.
