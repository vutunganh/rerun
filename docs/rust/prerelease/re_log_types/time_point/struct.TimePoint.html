pub struct TimePoint(BTreeMap<Timeline, TimeInt>);
Expand description

A point in time on any number of Timelines.

It can be represented by Time, a sequence index, or a mix of several things.

If this is empty, the data is timeless. Timeless data will show up on all timelines, past and future, and will hit all time queries. In other words, it is always there.
