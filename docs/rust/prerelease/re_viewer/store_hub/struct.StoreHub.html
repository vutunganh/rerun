Expand description

Interface for accessing all blueprints and recordings

The StoreHub provides access to the StoreDb instances that are used to store both blueprints and recordings.

Internally, the StoreHub tracks which ApplicationId and recording id (StoreId) are currently selected in the viewer. These can be configured using StoreHub::set_recording_id and StoreHub::set_app_id respectively.
