pub fn spawn(opts: &SpawnOptions) -> Result<(), SpawnError>
Expand description

Spawns a new Rerun Viewer process ready to listen for TCP connections.

If there is already a process listening on this port (Rerun or not), this function returns Ok WITHOUT spawning a rerun process (!).

Refer to SpawnOptions’s documentation for configuration options.

This only starts a Viewer process: if you’d like to connect to it and start sending data, refer to crate::RecordingStream::connect or use crate::RecordingStream::spawn directly.
