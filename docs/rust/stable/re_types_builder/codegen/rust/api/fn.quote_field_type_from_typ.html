fn quote_field_type_from_typ(typ: &Type, unwrap: bool) -> (TokenStream, bool)
Expand description

Returns type name as string and whether it was force unwrapped.

Specifying unwrap = true will unwrap the final type before returning it, e.g. Vec<String> becomes just String. The returned boolean indicates whether there was anything to unwrap at all.
