20 /// **Component**: A way to filter a set of `EntityPath`s.
21 ///
22 /// This implements as simple set of include/exclude rules:
23 ///
24 /// ```diff
25 /// + /world/** # add everything…
26 /// - /world/roads/** # …but remove all roads…
27 /// + /world/roads/main # …but show main road
28 /// ```
29 ///
30 /// If there is multiple matching rules, the most specific rule wins.
31 /// If there are multiple rules of the same specificity, the last one wins.
32 /// If no rules match, the path is excluded.
33 ///
34 /// The `/**` suffix matches the whole subtree, i.e. self and any child, recursively
35 /// (`/world/**` matches both `/world` and `/world/car/driver`).
36 /// Other uses of `*` are not (yet) supported.
37 ///
38 /// `EntityPathFilter` sorts the rule by entity path, with recursive coming before non-recursive.
39 /// This means the last matching rule is also the most specific one.
40 /// For instance:
41 ///
42 /// ```diff
43 /// + /world/**
44 /// - /world
45 /// - /world/car/**
46 /// + /world/car/driver
47 /// ```
48 ///
49 /// The last rule matching `/world/car/driver` is `+ /world/car/driver`, so it is included.
50 /// The last rule matching `/world/car/hood` is `- /world/car/**`, so it is excluded.
51 /// The last rule matching `/world` is `- /world`, so it is excluded.
52 /// The last rule matching `/world/house` is `+ /world/**`, so it is included.
53 ///
54 /// Unstable. Used for the ongoing blueprint experimentations.
